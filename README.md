# PLN
PLNCompiler - компилятор изобретенного языка программирования PLN который направлен на обучение основам алгоритмизации с использованием русского синтаксиса. Разработка длилась на протяжении года, с сентября 2017 по июнь 2018 и была дипломным проектом, ярким и столь незаметным 

Примеры программ - PLNCompiler/doc

Справка по языку - PLNCompiler/doc/PLNСправка.chm

Компилятором предусмотрена возможность подлкючения сборок .Net по их короткому имени.
Для преобразования короткого имени в квалифицированное, используется файл libs.xml, который
должен быть сгенерирован классом GACGenerator. На данный момент, такой опции для компилятора нет,
поэтому должно быть сделано вручную. 

Все файлы, которые не являются частью исходного кода, 
но являются частью проекта лежат по PLNCompiler/doc, а именно:

1)GRABC.pas и GRABC.dll - графическая библиотка из комплекта PascalABC.Net

2)libs.xml - обеспечивает подключение сборок по короткому имени, желательно перегенерировать по
ранее приведенной инструкции, после установки

3)SmartGOTO.docx - заметки по реализации оператора GOTO

Исключение составляет компонент FastColoredTextBox - сторонняя разработка,
применяемая для подсветки синтаксиса, помещена в папку PLNEditor/dll.

**Сборка**

1) Сборка консольного копилятора осуществляется сборкой проекта PLNCompiler
2) Сборка редактора кода+компилятора осуществляется сборкой проекта PLNEditor
